# Installation

- install erlang
- install elixir (1.11.1 or later)
- install postgresql
- install pgroonga https://pgroonga.github.io/ja
- clone this project
- create .env
- run `mix deps.get`
- run `mix ecto.create`
- run `mix ecto.migrate`
- run `source .env && mix run --no-halt`
